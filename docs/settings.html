⚙️ 設定

📧 メール通知設定（Gmail）

アプリを開いた時に自動でスケジュールをメール送信します。
送信には EmailJS（無料・月200通）を使用します。

▶ EmailJS の初回設定手順（ここをタップ）
  1. emailjs.com にアクセスしてアカウント作成
  2. 「Email Services」→「Add New Service」→「Gmail」を選択して接続
  3. 「Email Templates」→「Create New Template」
    テンプレートの本文に {{message}} を入れて保存
  4. 「Account」→「General」にある Public Key をコピー
  5. Service ID・Template ID・Public Key を下に入力して保存
データ管理

データはこのデバイスのブラウザ（localStorage）に保存されています。
